# SpecialTopics_2026Spring_UAVs

This repository is designed for students who are interested in **UAV control, robotics, and autonomous systems**.  
The goal is to build a solid foundation from **C++ programming → ROS2 → simulation → control systems → UAV applications**.

---

## 🚀 Course Objective

By the end of this course, students will be able to:

- Develop robotics applications using **C++**
- Understand and implement **ROS2-based systems**
- Build and test simulations in **Gazebo**
- Design and analyze **control algorithms**
- Apply control strategies to **UAV systems**

---

## 🎯 Expected Outcome

Students will be able to:

- Build a complete **ROS2-based UAV pipeline**
- Design controllers and validate them in simulation
- Understand how theory connects to real-world robotics systems

---

## 👨‍🏫 Instructor Notes

This repository is intended for **hands-on learning**.  
Students are encouraged to experiment, break things, and rebuild them.
